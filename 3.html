      인공지능의 종류

    1. 약인공지능

      2. 정해진 데이터와 규칙에 따라 작동하는 소프트웨어

      특정한 범위 내에서 활용 가능하며 큐칙을 벗어난 학습은 어려움

    2. 강인공지능

      3. 인공지능 스스로 데이터를 찾아 학습하여 범용으로 사용 가능

      인간의 지능과 비슷하거나 인간의 지능을 뛰어넘는 복잡한 사고와 추론도 가능

      현재 개발된 것은 아니며 상용화된 대부분은 약인공지능 수준

    3. 인공지능과 비인공지능

      4. 인공지능이 적용되지 않은 소프트웨어는 프로그래머가 작성한 명령을 자동화하여 작업을 빠르고 정확하게 반복 수행

      인공지능이 적용된 소프트웨어는 자동화뿐만 아니라 사람처럼 추론하고 예측, 데이터 기반 학습을 통해 성능 개선
